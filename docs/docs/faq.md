# 常见问题

## Q: 域名需要备案吗？

A: 使用 GitHub Pages 的免费域名不需要备案。如果使用国内域名，需要备案。

## Q: 网站访问速度如何？

A: GitHub Pages 使用全球 CDN，访问速度较快。国内可能稍慢，可使用 Gitee Pages。

## Q: 如何添加评论功能？

A: 推荐使用 Waline 或 Gitalk，配置简单，免费使用。

## Q: 能绑定自己的域名吗？

A: 可以，只需在 DNS 中添加 CNAME 记录指向 GitHub Pages。

## Q: 如何更新文章？

A: 在本地编辑 Markdown 文件，推送到 GitHub，自动更新。

## Q: 这个方案适合什么场景？

A: 个人博客、技术文档、项目展示等。不适合需要用户系统、复杂交互的网站。
